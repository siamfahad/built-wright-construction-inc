Built Wright Construction Inc. – Premium Web Portal
This is a high-performance, modern web application built for Built Wright Construction Inc., a premium construction and renovation company specializing in custom cabinetry and full-scale residential transformations across the Greater Toronto Area (GTA).

🏗️ About the Company
Built Wright Construction Inc. is dedicated to "Excellence in every square foot." Based in Ontario, the company provides high-standard construction services including:

Custom Cabinetry & Fabrication

Full Residential Renovations

Modern Interior Design & Installation

GTA-wide Service Coverage (Scarborough, Toronto, and surrounding regions)

🚀 Technical Architecture
The site is engineered for speed, security, and a "futuristic" user experience:

Framework: Next.js 15+ with App Router

Styling: Tailwind CSS with custom Glassmorphism and Neon Glow UI components

Animations: Sophisticated CSS-keyframe animations and interactive mobile-specific navigation

Media Management: Integrated Cloudinary for high-fidelity project gallery management

Icons: Lucide React

🛡️ Security Features
The application includes global security hardening:

Security Headers: Hardened next.config.ts with X-Frame-Options (Anti-Phishing/Clickjacking), HSTS, and Content-Type sniffing protection.

Bot Mitigation: Optimized metadata and honeypot structures.

🛠️ Getting Started
First, install dependencies and run the development server:

Bash
npm install
npm run dev
Open http://localhost:3000 with your browser to see the result.

📁 Project Structure Highlights
/app: Contains the main page logic, including specialized sections for Discovery, Fabrication, and Design.

/components: Reusable UI elements like the neon-glow FooterSection and interactive ContactInfoBox.

next.config.ts: Configuration for Cloudinary image domains and global security headers.
